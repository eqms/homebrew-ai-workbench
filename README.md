# homebrew-ai-workbench

Homebrew tap for [AI Workbench](https://github.com/eqms/ai-workbench) — a
Rust-based TUI multiplexer for AI-assisted development.

## Install

```bash
brew install eqms/ai-workbench/ai-workbench
```

## Update

```bash
brew update
brew upgrade ai-workbench
```

The formula is updated automatically by the release pipeline in
[eqms/ai-workbench](https://github.com/eqms/ai-workbench) on every tagged
release.
